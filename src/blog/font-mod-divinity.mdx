# 《神界：原罪 2》简体中文字体替换补丁

> 替换 **Definitive Edition** 版本内置中文字体为微软雅黑（Version 11.3.0）及方正新楷体。

### 使用

1. 使用 `Game.pak` 替换游戏根目录下 `\DefEd\Data\Game.pak`

2. 使用 `ChineseData.pak` 替换游戏根目录下 `\DefEd\Data\Localization\Chinese\ChineseData.pak`
